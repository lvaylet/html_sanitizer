In the Palette you see that the component tFileInputDelimited is saved in the folder File and the subfolder Input.

Note: There are 4 different types of components.

- Input component. Reads data from a source (external or not) and returns it in the flow connection.

-  Process component. Receives data, makes a specific transformation, and returns the transformed data in the flow connection.

- Output component. Receives the data from a flow and outputs it into a target.

- Virtual component. Made of two or more other components.

  1. After entering all details the final schema should look like this and you click OK:

  1. The system will ask you if you want to propagate the changes. You click Yes:

    Note: If you have modified the context variable from repository or the schema like in this lab for example, you are required to propagate the changes to the Job which is using this schema or variable.
